## How to run the projects

1. Make sure that you have `Node.js` intalled
    - Access [Node.js](https://nodejs.org/) and download the last stable version
2. Open your favorite terminal and navigate to some project folder
3. Run `npm install`
4. Run `npm run web`
5. Enjoy!
